# Spring boot test annotation
> @Test
>> java 소스에 대한 테스트 실행

# Spring boot 통합 테스트
> @SpringBootTest 
>> 스프링 컨테이너와 테스트를 함께 실행


> @Transactional
>> 테스트 케이스에 @Transactional이 있으면 테스트 시작 전에 트랜잭션을 시작하고 테스트 완료 후에 롤백한다. DB에 영향을 주지 않음.
