# 주로 쓰는 IntelliJ 단축키(Window 기준)

> generate : getter/setter/생성자 자동완성 - Alt + Insert  
> @Test : JUnit 테스트 annotation, 테스트 클래서 생성 - Ctrl + Shift + T  
> 맨끝 자동완성("", ;) : Ctrl + Shift + Enter  
> 파일 이동 : Ctrl + E
> 변수 생성 : Ctrl + Alt + V
> 복사 붙여 : Ctrl + D
> 에러 부분으로 이동 : F2
> 해당 글 선택 : Ctrl + W
> ReName : Shift + F6
